## Special variables and Constants

| Name  | Meaning                                    |
| :---: | ------------------------------------------ |
|  ans  | Most recent answer.                        |
|  eps  | Accuracy of floating-point precision.      |
|  i,j  | The imaginary unit √-1.                    |
|  Inf  | Infinity.                                  |
|  NaN  | Undefined numerical result (not a number). |
|  pi   | The number π                               |
